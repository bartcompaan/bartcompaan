# Bartcompaan — DJ site (snel live)

## 1) Pas je contactgegevens aan
Open `index.html` en vervang:
- `bartcompaan@example.com`
- Instagram / telefoon links (zoek op `@bartcompaan` en `+31 6`)

## 2) Pas je embeds aan
In `index.html`:
- Zoek `SOUNDCLOUD_EMBED` en vervang de SoundCloud URL met jouw profiel of mix.

## 3) Online zetten (3 makkelijke opties)
### A) Netlify (makkelijkst)
1. Ga naar Netlify > Add new site > Deploy manually
2. Sleep de hele map (of de zip uitpakken en dan de map uploaden)
3. Klaar — je krijgt direct een live link.

### B) GitHub Pages
1. Maak repo `bartcompaan`
2. Upload alle bestanden
3. Settings > Pages > Deploy from branch (main / root)

### C) Vercel
1. New Project > import repo
2. Deploy

## 4) Domein koppelen
Bij Netlify/Vercel: Domains > add custom domain `bartcompaan.nl` (of wat je kiest).
